# PersonRegistryManager

## Készítsen egy személyeket nyilvántartó, karbantartó Spring Boot alkalmazást. 

A feladathoz tartozó adatbázis tartalmazzon 3 táblát. (Személyek, címek, elérhetőségek) 
Egy személynek maximum kettő címe lehet. (állandó és ideiglenes) 
Egy címhez több elérhetőség (email, telefon, stb.) tartozhat. 

Az alkalmazás feladatai: 
- Adatok lekérdezése 
- Adatok rögzítése, módosítása, törlése, hibakezeléssel 
- Tesztesetek írása
